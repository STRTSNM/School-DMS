# School-DMS
Yet another school data management system

# How-to-run:
1. cd into main folder
2. run `python manage.py runserver`
3. Veiw the rendered page at localhost:8000

# Requirements:
1. Python 3.7+
2. MariaDB
3. Django
4. Pandas
5. Openpyxl
